# File To Video Converer

An Open Source Converter Bot scrapped from [Anydl Repo](https://github.com/spechide/anydlbot)



## Installation


### The Easy Way

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/kuanidonya/ConvertToVideo)

### The Normal Way

```
git clone https://github.com/kuanidonya/ConvertToVideo
cd ConvertToVideo
virtualenv -p python3 VENV
. ./VENV/bin/activate
pip install -r requirements.txt
python3 bot.py
```
